# Harmonix Mobile

Mobile-first rewrite of Harmonix using Expo, React Native, TypeScript, and InsForge as the backend platform.

## Stack

- Mobile: Expo + React Native + TypeScript + Expo Router
- Backend: InsForge Auth, Database, Storage, Functions, and MCP/CLI tooling
- Shared contracts: `packages/shared`

The repo is intentionally focused on the mobile client and InsForge setup.

## Commands

```bash
pnpm install
pnpm dev:mobile
pnpm typecheck
```

## Workspace Layout

```text
apps/mobile        Expo mobile app
packages/shared    Shared music/source types and schemas
docs/              InsForge backend notes
AGENTS.md          InsForge MCP guidance generated by setup
```
